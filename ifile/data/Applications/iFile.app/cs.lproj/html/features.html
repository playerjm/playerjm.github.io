• iFile:

    • Je prohlížeč souborů s podporou
      • Vytváření, mazání, kopírování, vyjímání (přemisťování), vkládání nebo zipování více souborů nebo adresářů v jednom
      • Přejmenovávání souborů nebo adresářů
      • "Záložkování" souborů a složek
      • Hledání souborů rekurzivně
      • Posílání jednoho, či více souborů přes Apple MobileMail
      • Editace textových souborů, HTML souborů, seznam vlastostí souborů, binárních souborů (s Hex Editorem) a SQLite3 databází souborů.
      • Editování vlastností souborů (jméno, vlastník, skupina, oprávnění, datum změny)
      • Průchod souboru do dalších AppStore aplikací
      • Otevření souborů a různých typů souborů pomocí dalších aplikací v iFile
    • Až do 8 souborů prohlizeče.
    • Plná podpora pro Dropbox/Box/FTP/WebDAV client: upload, download, kopírování, přesunutí, přejmenování, miniatury, atd.
    • Toshiba FlashAir® Podpora pro download souborů do lokálního souborového systému.
    • Má integovyný webový server s možností nahrávat a stahovat do iPhone/iPodu soubory
    • WebDAV podpora Web Serveru pro použití iFile jako sítového disku s přístupem pomocí iFile's WebDAV client.
    • Přenos souborů pomocí Bluetooth do iFile na dalším zařízení.
    • Přenos souborů pomocí Celeste a AirBlue sdílení do jiného zařízení
    • Import skladeb, videí a knih do iPod knihovny pomocí Gremlin knihovny.
    • Instalace Debian (deb) balíčků
    • Obsahuje prohlížeče souborů s podporou těchto typů:
      • Zvukové: aac, aiff, aif, aifc, amr, alac, caf, m4a, m4r, mp2, mp3, mpga, pcm, snd, wav, wma
      • Textové: c, cfg, cnf, conf, cpp, css, h, j, java, js, list, log, m, nib, php, plist, script, sh, strings, txt, xib
      • Video: mov, mp4, mpv, m4v, 3gp, 3g2, 3gpp2
      • Obrázky: gif, jpeg, jpg, png, tiff, tif, bmp, crw, cr2, erf, raf, 3fr, kdc, mef, mrw, nrw, nef, orf, rw2, pef, arw, sr2, svg
      • Dokumenty: rtf, html, htm, pdf
      • Soubory Microsoftu: doc, docx, xls, xlsx, ppt, pps, pptx
      • Soubory Apple iWork: pages, numbers, key
      • SQLite3 Database Typů Souborů: sqlite, sqlite3, sqlitedb, db.
      • Ostatní: zip, 7z, deb, rar, tar, tar.gz, tar.bz2, tar.Z, tgz, tbz2, gz, bz2, Z, xml

  • Omezené vlastnosti:

    Tyto funkce iFile jsou dostupné JEN registrovaným uživatelům:

    • Otevírání souborů v iFile z jiné aplikace použitím URL schématu ifile://
    • Hledání souborů
    • Podpora externího prohlížeče
    • Přehrávaní seznamů skladeb v novém přehrávači zvuku
    • Zobrazení názvu skladby, narozdíl od jména souboru
    • Zobrazení miniatur, namísto standardní ikony obrázku
    • Funkce hledání v editoru textu
    • WebDAV podpora iFile's Web Server.
    • Odesílání souborů pomocí Bluetooth do iFile na jiném zařízení.
    • použití vícenásobných tabulek.
    • Podpora FTP, box, Dropbox, WebDAV clientů.
    • Editace SQLite3 soborů database.
    • Ukládání modifikovaných binarních souborů z Hex Editor.
