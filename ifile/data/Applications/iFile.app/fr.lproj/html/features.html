• Fonctionalités de iFile

    • Gestionnaire de fichiers intégré avec comme caractéristiques
      • Création, suppression, copie, déplacement, coller ou zippage de multiples fichiers ou répertoires en une seule fois
      • Renomme les fichiers ou les dossiers
      • Ajout des Signet et des dossiers de signets
      • Recherche de fichers hiérarchiquement
      • Envoi par mail de un ou plusieurs fichiers avec Apple MobileMail
      • Edite des fichiers texte, HTML,fichier list de préférences, fichier binaire (avec l'éditeur Hexa ) et fichier de base de données SQLite3.
      • Edition des droits (noms, propriétaire, groupe, permissions, date de modifications)
      • Fichier d'autres apps venant de AppStore
      • Ouverture de divers fichiers provenant de nombreuses app directement depuis iFile
    • Jusqu'à 8 onglets fichiers.
    • Client avec support intégral de Dropbox/Box/FTP/WebDAV: téléchargement, téléversement, copie, déplacement, renommage, vignettage etc.
    • Support de Toshiba FlashAir® pour téléchargement en local.
    • Serveur Web intégré avec la possibilité de télécharger des fichiers ou de parcourir le contenu des dossiers.
    • Support WebDAV des Web Server pour utiliser iFile comme disque réseau et accéder en WebDav avec le client iFile.
    • Transfert de fichier par Bluetooth depuis iFile sur différents périphériques.
    • Transfert de fichier avec Celeste et le partage AirBlue vers un autre périphérique.
    • Import de chansons, videos et livres dans la bibliothèque de l'iPod depuis la librairie Gremlin.
    • Installation de packages Debian (deb)
    • Visualisateurs avec le support d'un grand nombre de formats :
      • Sons de types: aac, aiff, aif, aifc, amr, alac, caf, m4a, m4r, mp2, mp3, mpga, pcm, snd, wav, wma
      • Fichiers texte de types: c, cfg, cnf, conf, cpp, css, h, j, java, js, list, log, m, nib, php, plist, script, sh, strings, txt, xib
      • Films de types: mov, mp4, mpv, m4v, 3gp, 3g2, 3gpp2
      • Images de types: gif, jpeg, jpg, png, tiff, tif, bmp, crw, cr2, erf, raf, 3fr, kdc, mef, mrw, nrw, nef, orf, rw2, pef, arw, sr2, svg
      • Document de types: rtf, html, htm, pdf
      • Fichiers MicroSoft office de type: doc, docx, xls, xlsx, ppt, pps, pptx
      • Fichiers Apple iWork de type: pages, numbers, key
      • Fichier base de donnée SQLite3 au formats: sqlite, sqlite3, sqlitedb, db.
      • Autres formats: zip, 7z, deb, rar, tar, tar.gz, tar.bz2, tar.Z, tgz, tbz2, gz, bz2, Z, xml

  • Fonctions Limitées

    Quelques fonctions de iFile sont disponibles uniquement aux utilisateurs enregistrés:

    • Ouvrir des fichiers dans iFile depuis une autre application en utilisant le chemin ifile://.
    • Recherche de fichiers par nom.
    • Support de visaulisateur externe.
    • Jouer des Playlist depuis le nouveau Player Audio.
    • Voir le tiitre des musiques à la place du nom des fichiers.
    • Voir des vignettes des images à la place d'une icône.
    • Fonctionnalités de l'éditeur de recherche.
    • Support de WebDAV avec le serveur iFile.
    • Envoi de fichier en Bluetooth depuis iFile vers un autre périphérique.
    • Utilisation de plusieurs onglets.
    • Support FTP, box, Dropbox, Client WebDAV.
    • Edition de fichiers base de données SQLite3.
    • Sauvegarde des fichiers binaires modifiés depuis d'éditeur Hexa.
