• Caracteristici ale iFile

    • Navigatorul integrat de fișiere cu suport pentru
      • Crearea, ștergerea, copierea, mutarea, lipirea sau crearea de fișiere zip din fișiere multiple sau directoare dintr-o dată
      • Redenumirea de fișiere sau directoare
      • Semne de carte ale fișierelor și a directoarelor
      • Căutarea fișierelor ierarhic
      • Trimiterea de un sau mai multe fișiere prin email cu Apple MobileMail
      • Editarea fisierelor text, HTML, fisierele tip property list, fisierele binare (folosind editorul Hex) si bazele de date SQLite3.
      • Editarea atributelor fișierelor (nume, proprietar, grup, permisiuni, data de modificare)
      • Deschiderea fisierelor in alte aplicatii din AppStore
      • Deschiderea fisierelor din alte aplicatii in iFile
    • Pana la 8 taburi in browserul integrat.
    • Suport pentry Dropbox/Box/FTP/WebDAV: upload, download, copiere, mutare, redenumire, icoane reprezentative etc.
    • Suport pentru Toshiba FlashAir® prin care puteti descarca fisiere local.
    • Server Web integrat cu capabilități de a încărca și a descărca fișiere la directorul curent din navigatorul de fișiere
    • Suport pentru WebDAV incluzand crearea unui server Web pentru utilizarea in iFile ca un hard disk in retea.
    • Transferul fisierelor prin Bluetooth intre device-uri.
    • Transfer de fisiere folosind Celeste si AirBlue Sharing catre un alt device
    • Importarea melodiilor, videoclipurilor sau cartilor in libraria aplicatiei iPod folosind libraria Gremlin (este necesara achizitionarea Celeste sau YourTube 2/3/HD)
    • Instalare de pachete Debian (deb)
    • Vizualizatoarele de fișiere cu suport pentu următoarele fișiere
      • Tipuri de fișiere Audio: aac, aiff, aif, aifc, amr, alac, caf, m4a, m4r, mp2, mp3, mpga, pcm, snd, wav, wma
      • Tipuri de fișiere Text: c, cfg, cnf, conf, cpp, css, h, j, java, js, list, log, m, mm, nib, php, plist, script, sh, strings, txt, xib
      • Tipuri de fișiere Video: mov, mp4, mpv, m4v, 3gp, 3g2, 3gpp2
      • Tipuri de fișiere Imagini: gif, jpeg, jpg, png, tiff, tif, bmp, crw, cr2, erf, raf, 3fr, kdc, mef, mrw, nrw, nef, orf, rw2, pef, arw, sr2, svg
      • Tipuri de fișiere Documente: rtf, html, htm, pdf
      • Tipuri de fișiere Microsoft: doc, docx, xls, xlsx, ppt, pps, pptx
      • Tipuri de fișiere Apple iWork: pages, numbers, key
      • Bazele de date SQLite3 suportate: sqlite, sqlite3, sqlitedb, db.
      • Alte tipuri de fișiere: zip, 7z, deb, rar, tar, tar.gz, tar.bz2, tar.Z, tgz, tbz2, gz, bz2, Z, xml

  • Caracteristici restricționate

    Anumite funcționalități ale iFile sunt disponibile doar utilizatorilor înregistrați:

    • Căutare fișier după nume.
    • Suport pentru vizualizator extern.
    • Listă de redare rulând în noul Player Audio.
    • Afișând titlul cântecului înloc de numele fișierului pentru fișierele audio.
    • Afișând miniaturi înloc de iconițele implicite pentru imagini.
    • Funcționalitatea de-a căuta în editorul de text.
    • Suport WebDAV pentru web serverul iFile.
    • Trimiterea fisierelor prin Bluetooth intre deviceuri.
    • Utilizarea mai multor taburi simultan.
    • Support pentru FTP, box, Dropbox, WebDAV.
    • Editarea bazelor de date SQLite3.
    • Salvarea fisierelor binare folosind Editorul Hex.
