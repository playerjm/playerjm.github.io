• Funzioni

    • Esploratore file integrato con il supporto per
      • Creazione, eliminazione, copia, taglia (sposta) e incolla o compressione (ZIP) di file multipli o cartelle in una sola volta
      • Rinominazione di file o cartelle
      • Preferiti di file o cartelle
      • Ricerca file gerarchicamente
      • Invio via email di uno o più file tramite Apple Mail
      • Modifica di file di testo, HTML, liste proprietà, binari (con Hex Editor) e database SQLite3.
      • Modifica degli attributi dei file (nome, proprietario, gruppo, permessi e data di modifica)
      • Passaggio di file ad applicazioni AppStore
      • Apertura di file Opening of files of various file types from other apps in iFile
    • Fino a 8 schede dell'esploratore file.
    • Supporto completo a Dropbox/Box/FTP/WebDAV: invio, ricezione, copia, sposta, rinomina, anteprime ecc.
    • Supporto Toshiba FlashAir® per scaricare file in locale.
    • Server WEB integrato con la possibilità di inviare e scaricare file dal percorso attuale dell'esploratore file.
    • Supporto WebDAV al Server WEB per utilizzare iFile come un disco di rete a cui accedere tramite il client WebDAV di iFile.
    • Invio di file ad altri dispositivi con iFile tramite Bluetooth.
    • Invio di file ad altri dispositivi tramite Celeste e AirBlue Sharing.
    • Importazione di musica, video e libri nella libreria iPod tramite libreria Gremlin.
    • Installazione di pacchetti Debian (deb)
    • Visualizzatori file con il supporto per i seguenti formati
      • Audio: aac, aiff, aif, aifc, amr, alac, caf, m4a, m4r, mp2, mp3, mpga, pcm, snd, wav, wma
      • Testo: c, cfg, cnf, conf, cpp, css, h, j, java, js, list, log, m, nib, php, plist, script, sh, strings, txt, xib
      • Video: mov, mp4, mpv, m4v, 3gp, 3g2, 3gpp2
      • Immagini: gif, jpeg, jpg, png, tiff, tif, bmp, crw, cr2, erf, raf, 3fr, kdc, mef, mrw, nrw, nef, orf, rw2, pef, arw, sr2, svg
      • Documenti: rtf, html, htm, pdf
      • Microsoft: doc, docx, xls, xlsx, ppt, pps, pptx
      • Apple iWork: pages, numbers, key
      • Tipi di file database SQLite3: sqlite, sqlite3, sqlitedb, db.
      • Altro: zip, 7z, deb, rar, tar, tar.gz, tar.bz2, tar.Z, tgz, tbz2, gz, bz2, Z, xml

  • Funzioni ristrette

    Alcune funzioni di iFile sono disponibili solo per gli utenti registrati:

    • Apertura di file in iFile da altre applicazioni utilizzando lo schema URL ifile://.
    • Ricerca file.
    • Supporto ai visualizzatori esterni.
    • Riproduzione di playlist nel nuovo player audio.
    • Visualizzazione dei titoli delle canzoni invece dei nome dei file audio.
    • Visualizzazione delle anteprime invece dell'icona predefinita per le immagini.
    • Ricerca nel visualizzatore testi.
    • Supporto WebDAV al Server WEB di iFile.
    • Invio di file ad altri dispositivi con iFile tramite Bluetooth.
    • Utilizzo di schede multiple.
    • Supporto ad FTP, box, Dropbox, WebDAV.
    • Modifica dei file database SQLite3.
    • Salvataggio dei file binari modificati con Hex Editor.
