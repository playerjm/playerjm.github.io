• Возможности программы iFile

    • Встроенный файловый менеджер позволяет:
      • Создавать, удалять, копировать, перемещать файлы, создавать ZIP-архивы с несколькими файлами и папками.
      • Переименовывать файлы и папки.
      • Создавать закладки для быстрого доступа к файлам или папкам.
      • Искать файлы по нисходящей (иерархически).
      • Отправлять любые файлы электронной почтой, используя стандартную почтовую программу.
      • Редактирование текстовых, HTML, PLIST-файлов, двоичных файлов (с помощью Hex-редактора) и файлов баз данных SQLite3.
      • Изменять атрибуты (имя, владелец, группа, права доступа, дата изменений)
      • Передача файлов другим программам из AppStore.
      • Открытие файлов различных типов из других программ в iFile.
    • До 8 вкладок в Менеджере файлов.
    • Полная поддержка клиентов Dropbox/Box/FTP/WebDAV: загрузка на сервер, скачивание, копирование, перемещение, переименование, эскизы страниц и т.д.
    • Поддержка Toshiba FlashAir® для скачивания файлов в локальную файловую систему.
    • Встроенный веб-сервер с возможностью скачивать/закачивать файлы и папки.
    • Поддержка WebDAV Веб-сервером, чтобы использовать iFile как сетевой диск и обеспечивать доступ клиентам WebDAV.
    • Передача файлов по Bluetooth в программу iFile на другом устройстве.
    • Передача файлов через Celeste и AirBlue Sharing на другое устройство.
    • Импорт песен, видео и книг в медиатеку iPod через библиотеку Gremlin.
    • Установщик пакетов Debian (deb) (программы Cydia)
    • Просмотр файлов следующих типов
      • Аудио: aac, aiff, aif, aifc, amr, alac, caf, m4a, m4r, mp2, mp3, mpga, pcm, snd, wav, wma
      • Текст: c, cfg, cnf, conf, cpp, css, h, j, java, js, list, log, m, nib, php, plist, script, sh, strings, txt, xib
      • Видео: mov, mp4, mpv, m4v, 3gp, 3g2, 3gpp2
      • Изображения: gif, jpeg, jpg, png, tiff, tif, bmp, crw, cr2, erf, raf, 3fr, kdc, mef, mrw, nrw, nef, orf, rw2, pef, arw, sr2, svg
      • Документы: rtf, html, htm, pdf
      • Документы Microsoft Office: doc, docx, xls, xlsx, ppt, pps, pptx
      • Документы Apple iWork: pages, numbers, key
      • Базы данных SQLite3: sqlite, sqlite3, sqlitedb, db.
      • Другие: zip, 7z, deb, rar, tar, tar.gz, tar.bz2, tar.Z, tgz, tbz2, gz, bz2, Z, xml

  • Ограниченные функции

    Некоторые возможности iFile доступны только зарегистрированным пользователям:

    • Открытие файлов в iFile из других программ, используя протокол ifile://.
    • Поиск по имени файла.
    • Поддержка внешних средств просмотра.
    • Воспроизведение плейлистов в обновлённом Аудиопроигрывателе.
    • Отображение названий песен вместо названий их файлов.
    • Отображение мини-копий картинок вместо однородных значков.
    • Поиск в текстовом редакторе.
    • Поддержка WebDAV встроенным веб-сервером iFile.
    • Отправка файлов по Bluetooth в программу iFile на других устройствах.
    • Использование многочисленных вкладок.
    • Клиентская поддержка FTP, box, Dropbox, WebDAV.
    • Редактирование баз данных SQLite3.
    • Сохранение изменённых двоичных файлов в Hex-редакторе.
