• iFileの特徴

    • ファイルブラウザは以下の機能をサポートします
      • ファイル、フォルダ、およびシンボリックリンクを作成出来ます。
      • 複数のファイルやフォルダを一度に削除、コピー、カット（移動）、ペースト、およびZIP圧縮/解凍ができます。
      • ファイルやフォルダをブックマークすることができます。
      • 階層を辿ってファイル検索ができます。
      • ファイルやフォルダの名前を変えることができます。
      • 複数のファイルをメールに添付して送ることができます。
      • テキストファイル、HTMLファイル、プロパティリストファイル、バイナリファイル（Hexエディタ）とSQLite3のデータベースファイルの編集。
      • ファイル属性 (名前、オーナー、グループ、アクセス権、変更日時）を編集することができます。
      • 他のAppStoreアプリにファイルを渡す。
      • 他のアプリからさまざまな形式のファイルを開く。
    • ファイルブラウザのタブを最大8つに。
    • 完全なDropbox/Box/FTP/WebDAVクライアント機能: アップロード, ダウンロード, コピー, 移動, リネーム, サムネイル 等。
    • ローカルにファイルのダウンロードするためのToshiba FlashAir®をサポート
    • ファイルブラウザで表示中のフォルダをWebサーバーに公開し、ファイルのアップロード、ダウンロードができるようになります。
    • iFileをネットワークボリュームにするためのウェブサーバー機能、およびWebDAVクライアントを使用してアクセスするためのWebDAV機能をサポート。
    • Bluetooth経由で他のデバイス上のiFileにファイルを転送。
    • Celeste と AirBlue を使った他デバイスとのファイル共有。
    • Gremlin経由でiPodのライブラリに曲、ビデオ、ブックのインポート。
    • Debian パッケージ (deb) のインストール。
    • ファイルビューアーは以下のファイルタイプをサポートします。
      • サウンド: aac, aiff, aif, aifc, amr, alac, caf, m4a, m4r, mp2, mp3, mpga, pcm, snd, wav, wma
      • テキスト: c, cfg, cnf, conf, cpp, css, h, j, java, js, list, log, m, nib, php, plist, script, sh, strings, txt, xib
      • ムービー: mov, mp4, mpv, m4v, 3gp, 3g2, 3gpp2
      • 画像: gif, jpeg, jpg, png, tiff, tif, bmp, crw, cr2, erf, raf, 3fr, kdc, mef, mrw, nrw, nef, orf, rw2, pef, arw, sr2, svg
      • ドキュメント: rtf, html, htm, pdf
      • Microsoftファイル: doc, docx, xls, xlsx, ppt, pps, pptx
      • Apple iWorkファイル: pages, numbers, key
      • SQLite3データベースのファイルタイプ: sqlite, sqlite3, sqlitedb, db.
      • その他: zip, 7z, deb, rar, tar, tar.gz, tar.bz2, tar.Z, tgz, tbz2, gz, bz2, Z, xml

  • 追加機能

    iFileをレジストすると以下の機能が使えるようになります:

    • ”ifile://”というURLスキームを使うことで他のアプリケーションからiFileでファイルを開くようにできます。
    • ファイル名による検索。
    • 外部ビューアーへの対応。
    • プレイリストに対応した新しいオーディオプレイヤー
    • オーディオファイルの曲名を表示
    • 画像のサムネイルを表示
    • テキストエディタの検索機能。
    • ウェブサーバーのWebDAVをサポート。
    • 他のデバイス上のiFileにBluetooth経由でファイルを送信。
    • 複数タブの使用。
    • FTP, box, Dropbox, WebDAV クライアントをサポート。
    • SQLite3データベースの編集。
    • Hexエディタでバイナリファイルの修正・保存。
