• Eigenschappen van iFile

    • Geïntegreerde File Browser met ondersteuning voor:
      • Creëren, verwijderen, kopiëren, knippen (verplaatsen), plakken of zippen van meerdere bestanden of directories in een keer
      • Hernoemen van bestanden of directories
      • Bladwijzers voor bestanden en mappen
      • Bestanden zoeken
      • E-mailen van één of meerdere bestanden met Apple MobileMail
      • Bewerken van tekst bestanden, HTML bestanden, property list bestanden, binary bestanden (met de Hex Editor) en SQLite3 database bestanden.
      • Bewerken van bestandseigenschappen (naam, eigenaar, groep, permissies, modificatie datum)
      • Bestanden doorgeven naar andere AppStore applicaties
      • Openen van bestanden van verschillende bestandstypes uit andere apps in iFile
    • Tot 8 bestandverkenner pagina's.
    • Volledige Dropbox/Box/FTP/WebDAV client ondersteuning: upload, download, kopiëren, verplaatsen, hernoemen, pictogrammen etc.
    • Toshiba FlashAir® ondersteuning voor het downloaden van files naar het lokale bestandssysteem.
    • Geïntegreerde Web Server met download en upload mogelijkheden.
    • WebDAV ondersteuning van Web Server om iFile te gebruiken als netwerkschijf en om toegangkelijk te zijn in iFile's WebDAV client.
    • Bestandsoverdracht via Bluetooth naar iFile op een ander toestel.
    • Bestandsoverdracht via Celeste en AirBlue Sharing naar een ander toestel.
    • Importeren van nummers, videos en boeken naar de iPod bibiotheek via Gremlin Library.
    • Installatie van Debian pakketten (deb)
    • File Viewers ondersteunen de volgende bestand formaten:
      • Geluid bestandstypes: aac, aiff, aif, aifc, amr, alac, caf, m4a, m4r, mp2, mp3, mpga, pcm, snd, wav, wma
      • Tekst bestandstypes: c, cfg, cnf, conf, cpp, css, h, j, java, js, list, log, m, nib, php, plist, script, sh, strings, txt, xib
      • Film bestandstypes: mov, mp4, mpv, m4v, 3gp, 3g2, 3gpp2
      • Afbeelding bestandstypes: gif, jpeg, jpg, png, tiff, tif, bmp, crw, cr2, erf, raf, 3fr, kdc, mef, mrw, nrw, nef, orf, rw2, pef, arw, sr2, svg
      • Document bestandstypes: rtf, html, htm, pdf
      • Microsoft bestandstypes: doc, docx, xls, xlsx, ppt, pps, pptx
      • Apple iWork bestandstypes: pages, numbers, key
      • SQLite3 Database bestandstypes: sqlite, sqlite3, sqlitedb, db.
      • Andere bestandstypes: zip, 7z, deb, rar, tar, tar.gz, tar.bz2, tar.Z, tgz, tbz2, gz, bz2, Z, xml

  • Beperkte Eigenschappen

    Sommige functies zijn alleen beschikbaar voor geregistreerde gebruikers:

    • Bestanden openen in iFile via andere applicaties via het URL schema ifile://.
    • Zoeken op bestandsnaam.
    • Externe Viewer ondersteuning.
    • Afspeellijsten afspelen in de nieuwe Audio Player.
    • Het weergeven van de titel van het nummer in plaats van de naam van het bestand.
    • Het weergeven van miniatuurweergave voor afbeeldingen in plaats van de standaard iconen.
    • Zoek functionaliteit in de Tekst Editor.
    • WebDAV ondersteuning iFile's Web Server.
    • Versturen van bestanden via Bluetooth naar iFile op andere toestellen.
    • Gebruik van verschillende tabbladen.
    • FTP, box, Dropbox, WebDAV client ondersteuning.
    • Wijzigen van SQLite3 database bestanden.
    • Wijzigen van aangepaste binary bestanden vanuit de Hex Editor.
