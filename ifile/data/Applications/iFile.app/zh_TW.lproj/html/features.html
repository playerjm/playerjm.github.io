• iFile 的主要功能

    • 檔案瀏覽器，支援:
      • 同時新增、刪除、複製、剪下（移動）、貼上或壓縮多個檔案或資料夾；
      • 重新命名檔案或目錄；
      • 使用書簽標記檔案或檔案夾；
      • 跨層級搜尋檔案；
      • 使用蘋果 Mail 通過電子郵件發送一個或多個檔案；
      • 編輯文件、HTML、屬性表、二進制（使用十六進位編輯器）以及 SQLite3 資料庫等類型的檔案；
      • 編輯檔案屬性（名稱、擁有者、組別、權限和修改日期）。
      • 傳遞檔案給 App Store 應用程式使用；
      • 打開其它應用程式傳遞給 iFile 的多種類型的檔案。
    • 最多可使用 8 個浏覽標簽；
    • 完整的 Dropbox/Box/FTP/WebDAV 客戶端支援：上傳、下載、複制、移動、重命名、預覽圖等等；
    • 支持從東芝 FlashAir® 下載檔案到本地；
    • 網頁服務器，提供對檔案瀏覽器當前資料夾中檔案的上下載功能；
    • 網頁伺服器的 WebDAV 支持以將 iFile 作爲網絡硬碟使用或被 iFile 的 WebDAV 客戶端訪問；
    • 經藍牙傳輸檔案到其它裝置的 iFile；
    • 經 Celeste 和 AirBlue Sharing 傳輸檔案到其它裝置；
    • 使用 Gremlin 導入音樂、影片及電子書到 iPod 資源庫；
    • 安裝 Debian 封包（deb）；
    • 檔案查看器，支援下列類型：
      • 聲音檔案：aac、aiff、aif、aifc、amr、alac、caf、m4a、m4r、mp2、mp3、mpga、pcm、snd、wav、wma；
      • 純文字檔案：c、cfg、cnf、conf、cpp、css、h、j、java、js、list、log、m、mm、nib、php、plist、script、sh、strings、txt、xib；
      • 影片檔案：mov、mp4、mpv、m4v、3gp、3g2、3gpp2；
      • 圖片檔案：gif、jpeg、jpg、png、tiff、tif、bmp、crw、cr2、erf、raf、3fr、kdc、mef、mrw、nrw、nef、orf、rw2、pef、arw、sr2、svg；
      • 文檔檔案：rtf、html、htm、pdf、webarchive；
      • 微軟檔案：doc、docx、xls、xlsx、ppt、pps、pptx；
      • 蘋果 iWork 檔案：pages、numbers、key；
      • SQLite3 資料庫檔案類型：sqlite、sqlite3、sqlitedb、db；
      • 其他檔案：zip、7z、deb、rar、tar、tar.gz、tar.bz2、tar.Z、tgz、tbz2、gz、bz2、Z、xml。

  • 特級功能

    以下功能僅適用於已註冊者：

    • 檔案搜尋；
    • 播放曲目（Playlist）；
    • 瀏覽聲音檔時顯示歌名；
    • 瀏覽圖片時顯示縮圖；
    • 在純文字檔案裡搜尋；
    • iFile 網頁伺服器的 WebDAV 支援；
    • 使用 iFile 經藍牙傳輸檔案到其它裝置；
    • 使用多標簽；
    • FTP、box、Dropbox 及 WebDAV 客戶端支援；
    • 編輯 SQLite3 資料庫檔案；
    • 十六進位編輯器保存檔案。
