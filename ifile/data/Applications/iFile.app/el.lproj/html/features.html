• Χαρακτηριστικά του iFile

    • Ενσωματωμένος Περιηγητής Αρχείων με υποστήριξη για
      • Δημιουργία, διαγραφή, αντιγραφή, αποκοπή (μετακίνηση), επικκόληση και συμπίεση πολλαπλών αρχείων ή καταλόγων
      • Μετονομασία αρχείων και φακέλων/καταλόγων
      • Προσθήκη σελιδοδεικτών για αρχεία και φακέλους
      • Αναζήτηση αρχείων ιεραρχικά
      • Αποστολή Email με συννημένο ένα ή περισσότερα αρχεία με το MobileMail
      • Επεξεργασία αρχείων κειμένου, αρχείων HTML, αρχείων property list, αρχείων binary (με Hex Editor) και SQLite3 αρχείων.
      • Επεξεργασία γνωρισμάτων αρχείου (όνομα, ιδιοκτήτης, ομάδα, δικαιώματα, ημερομηνία διαμόρφωσης)
      • Άνοιγμα σε άλλες AppStore εφαρμογές
      • Άνοιγμα αρχείων από άλλες εφαρμογές στο iFile
    • Μέχρι και 8 file browser tabs.
    • Πλήρης υποστήριξη Dropbox/Box/FTP/WebDAV: ανέβασμα, κατέβασμα, αντιγραφή, μετακίνηση, μετονομασία, εικονίδια κλπ.
    • Υποστήριξη για Toshiba FlashAir® για κατέβασμα αρχείων στο τοπικό σύστημα αρχείων.
    • Υποστήριξη WebDAV του Διακομιστή Web για χρήση του iFile ως δίσκο δικτύου και να είναι προσβιβάσιμο με το iFile WebDAV client.
    • Αποστολή αρχείων με Bluetooth στο iFile άλλης συσκευής.
    • Αποστολή αρχείων μέσω Celeste και AirBlue Sharing σε άλλη συσκευή.
    • Εισαγωγή τραγουδιών, βίντεο και βιβλίων στο iPod library μέσω Gremlin library.
    • εγκατάσταση πακέτων Debian (deb)
    • Προβολείς αρχείων με υποστήριξη των παρακάτω τύπων αρχείων
      • Τύποι Αρχείων Ήχου: aac, aiff, aif, aifc, amr, alac, caf, m4a, m4r, mp2, mp3, mpga, pcm, snd, wav, wma
      • Τύποι Αρχείων Κειμένου: c, cfg, cnf, conf, cpp, css, h, j, java, js, list, log, m, nib, php, plist, script, sh, strings, txt, xib
      • Τύποι Αρχείων Ταινίας: mov, mp4, mpv, m4v, 3gp, 3g2, 3gpp2
      • Τύποι Αρχείων Εικόνας: gif, jpeg, jpg, png, tiff, tif, bmp, crw, cr2, erf, raf, 3fr, kdc, mef, mrw, nrw, nef, orf, rw2, pef, arw, sr2, svg
      • Τύποι Αρχείων Εγγράφων: rtf, html, htm, pdf
      • Τύποι Αρχείων Microsoft: doc, docx, xls, xlsx, ppt, pps, pptx
      • Τύποι Αρχείων Apple iWork: pages, numbers, key
      • SQLite3 Τύποι Αρχείου Βάσης Δεδομένων: sqlite, sqlite3, sqlitedb, db.
      • Άλλοι Τύποι Αρχείων: zip, 7z, deb, rar, tar, tar.gz, tar.bz2, tar.Z, tgz, tbz2, gz, bz2, Z, xml

  • Χαρακτηριστικά

    Κάποιες από τις λειτουργίες του iFile είναι διαθέσιμες μόνο σε εγγεγραμμένους χρήστες:

    • Άνοιγμα αρχείων στο iFile από άλλες εφαρμογές χρησιμοποιώντας το σχήμα URL ifile://.
    • Αναζήτηση Αρχείου.
    • Υποστήριξη Εξωτερικού Προβολέα.
    • Αναπαραγωγή Playlist στο νέο Player Μουσικής.
    • Εμφάνιση αυθεντικού τίτλου τραγουδιού αμτί για την κωδικοποιημένη ονομασία.
    • Εμφάνιση προεπισκοπίσεων εικόνων αντί για τα προεπιλεγμένε εικονίδια.
    • Αναζήτηση λέξεων σε κείμενο.
    • Υποστήριξη WebDAV στο iFile Web Server.
    • Αποστολή αρχείων μέσω Bluetoothστο iFile άλλης συσκευής.
    • Χρήση πολλαπλών tabs.
    • Υποστήριξη FTP, box, Dropbox, WebDAV client.
    • Επεξεργασία αρχείων SQLite3.
    • Αποθήκευση επεξεργασμένων αρχείων binary από τον Hex Editor.
