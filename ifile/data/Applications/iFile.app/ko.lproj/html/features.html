• iFile의 기능

    • 다양한 기능을 지원하는 통합 파일 브라우저
      • 파일 만들기, 삭제, 복사, 오려두기(이동하기), 붙이기 또는 여러개의 파일, 폴더를 한번에 압축하기
      • 파일 또는 폴더의 이름 변경
      • 파일과 폴더 북마크(즐겨찾기)
      • 파일 계층 검색
      • 애플 모바일 메일을 통해 하나 또는 그이상의 첨부 파일 전송
      • 텍스트 파일, HTML 파일, property list 파일, 이진 파일(Hex 편집기) 및 SQLite3 데이터베이스 파일 편집.
      • 파일 속성 편집(이름, 소유자, 그룹, 허가, 수정 날짜)
      • 다른 앱 스토어 애플리케이션으로 파일 전달
      • iFile에서 다른 애플리케이션의 다양한 형식의 파일 열기
    • 최대 8개의 파일 브라우저 탭.
    • 드롭박스/Box/FTP/WebDAV 클라이언드 완벽 지원: 업로드, 다운로드, 복사, 이동, 이름 변경, 미리보기 등등.
    • 로컬 파일 시스템으로 도시바 FlashAir® 파일 다운로드 지원.
    • 파일 브라우저에 존재하는 폴더로 파일 업로드, 다운로드가 가능한 통합 웹 서버
    • iFile을 네트워크 드라이브로 사용하며 iFile의 WebDAV 클라이언트로 접속할 수 있는 웹 서버 WebDAV 지원.
    • 블루투스를 통해 다른 기기의 iFile로 파일 전송.
    • Celeste와 AirBlue Sharing을 통해 다른 기기로 파일 전송.
    • Gremlin 라이브러리를 통해 노래, 비디오 및 책을 iPod 라리브러리로 가져오기 (Celeste 또는 YourTube 2/3/HD 구매 필요)
    • Debian 패키지 설치 (deb)
    • 아래의 다양한 형식을 지원하는 파일 뷰어
      • 사운드 파일 형식: aac, aiff, aif, aifc, amr, alac, caf, m4a, m4r, mp2, mp3, mpga, pcm, snd, wav, wma
      • 텍스트 파일 형식: c, cfg, cnf, conf, cpp, css, h, j, java, js, list, log, m, mm, nib, php, plist, script, sh, strings, txt, xib
      • 영상 파일 형식: mov, mp4, mpv, m4v, 3gp, 3g2, 3gpp2
      • 이미지 파일 형식: gif, jpeg, jpg, png, tiff, tif, bmp, crw, cr2, erf, raf, 3fr, kdc, mef, mrw, nrw, nef, orf, rw2, pef, arw, sr2, svg
      • 문서 파일 형식: rtf, html, htm, pdf
      • Microsoft 파일 형식: doc, docx, xls, xlsx, ppt, pps, pptx
      • 애플 iWork 파일 형식: pages, numbers, key
      • SQLite3 데이터베이스 파일 형식: sqlite, sqlite3, sqlitedb, db.
      • 기타 파일 형식: zip, 7z, deb, rar, tar, tar.gz, tar.bz2, tar.Z, tgz, tbz2, gz, bz2, Z, xml

  • 제한 기능

    iFile의 일부 기능은 등록 사용자만 가능합니다:

    • 파일 이름 검색.
    • 외부 뷰어 지원.
    • 새로운 오디오 플레이어의 재생 목록.
    • 오디오 파일의 파일명 대신 노래 제목 보기.
    • 이미지의 기본 아이콘 대신 이미지 미리보기.
    • 텍스트 편집기의 검색 기능.
    • iFile의 웹 서버 WebDAV 지원
    • 다른 기기의 iFile로 블루투스를 통해 파일 전송.
    • 여러개의 탭 사용.
    • FTP, box, 드롭박스, WebDAV 클라이언트 지원.
    • SQLite3 테이터베이스 파일 편집.
    • Hex 편집기를 사용한 이진 파일 수정 및 저장.
