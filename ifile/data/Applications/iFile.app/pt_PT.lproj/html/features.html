• Características do iFile

    • Explorador de ficheiros integrado, com suporte para:
      • Criar, apagar, copiar, cortar (mover), colar ou comprimir (zip) múltiplos ficheiros ou diretórios de uma vez
      • Renomear ficheiros ou diretórios
      • Adicionar marcadores de ficheiros e pastas
      • Pesquisa hierárquica de ficheiros
      • Enviar por email, um ou mais ficheiros, com Mobile Mail da Apple
      • Edição de ficheiros de texto, ficheiros HTML, ficheiros plist, ficheiros binários (com o Editor Hex) e ficheiros de base de dados SQLite3.
      • Editar atributos de ficheiros (nome, proprietário, grupo, permissão, data de modificação)
      • Passagem de ficheiros para outras apps da AppStore
      • Abrir ficheiros de vários tipos de outras apps no iFile
      • Passagem de ficheiros para outras apps da AppStore
      • Abertura de ficheiros de vários tipos de outras apps no iFile
    • Até um máximo de 8 tabuladores de gestão de ficheiros.
    • Suporte total de clientes Dropbox/Box/FTP/WebDAV: envio, descarga, cópia, mover, renomear, miniaturas etc.
    • Suporte Toshiba FlashAir® para descarregar ficheiros para o sistema de ficheiros local.
    • Até 8 tabuladores de explorador de ficheiros
    • Integração completa do Dropbox/FTP com suporte para copiar, mover, colar, criação diretórios, renomear e caching local
    • Servidor Web integrado com capacidade de descarregar (download) ou carregar (upload) ficheiros para o diretório selecionado no explorador de ficheiros.
    • Suporte WebDAV do Servidor Web para usar o iFile como drive de rede e ser acedido com o cliente WebDAV do iFile.
    • Transferência de ficheiros via Bluetooth para o iFile noutro dispositivo.
    • Transferência de ficheiros via Celeste e AirBlue Sharing para outro dispositivo.
    • Importação de músicas, vídeos e livros para a biblioteca iPod através da biblioteca Gremlin.
    • Instalação de pacotes Debian (deb)
    • Extração de vários tipos de ficheiros comprimidos, incluíndo ficheiros zip e rar protegidos por palavra-passe
    • Visualizador de ficheiros com suporte para os seguintes tipos:
      • Sons: aac, aiff, aif, aifc, amr, alac, caf, m4a, m4r, mp2, mp3, mpga, pcm, snd, wav, wma
      • Textos: c, cfg, cnf, conf, cpp, css, h, j, java, js, list, log, m, mm, nib, php, plist, script, sh, strings, txt, xib
      • Vídeos: mov, mp4, mpv, m4v, 3gp, 3g2, 3gpp2
      • Imagens: gif, jpeg, jpg, png, tiff, tif, bmp, crw, cr2, erf, raf, 3fr, kdc, mef, mrw, nrw, nef, orf, rw2, pef, arw, sr2, svg
      • Documentos: rtf, html, htm, pdf
      • Microsoft: doc, docx, xls, xlsx, ppt, pps, pptx
      • iWork da Apple: pages, numbers, key
      • SQLite3 Database File Types: sqlite, sqlite3, sqlitedb, db.
      • Outros: zip, 7z, deb, rar, tar, tar.gz, tar.bz2, tar.Z, tgz, tbz2, gz, bz2, Z, xml
    • Interação com outros Tweaks/Aplicações Cydia:
      • Safari Download Manager: Permite abrir ficheiros gravados diretamente no iFile.
      • AttachmentSaver: Permite abrir ficheiros gravados diretamente no iFile.
      • Music Controls: permite interagir com o Reprodutor de Áudio do iFile', p.ex. no Lock Screen, etc.

  • Características restritas

    Algumas funcionalidades do iFile apenas estão disponíveis para os utilizador registados:

    • Pesquisa de nomes de ficheiros
    • Reproduzir listas de músicas no novo Reprodutor de Áudio
    • Mostrar o título da música, em vez do nome do ficheiro de áudio
    • Mostrar miniaturas, em vez do ícone padrão para as imagens
    • Função de pesquisa no editor de textos
    • Suporte WebDAV do Servidor Web do iFile
    • Envio de ficheiros via Bluetooth para o iFile noutros dispositivos
    • Utilização de vários tabuladores
    • FTP, box, Dropbox, WebDAV client support.
    • Editing of SQLite3 database files.
    • Saving of modified binary files from Hex Editor.
