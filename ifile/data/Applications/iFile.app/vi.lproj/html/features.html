• Các tính năng của iFile:

    • Trình duyệt file tích hợp với hỗ trợ:
      • Tạo, xóa, sao chép, cắt (di chuyển), dán hoặc nén zip nhiều file hoặc thư mục cùng lúc
      • Đổi tên file hoặc thư mục
      • Tạo Bookmark file hoặc thư mục
      • Tìm kiếm file phân cấp
      • Gửi Email một hoặc nhiều file với Apple MobileMail
      • Soạn thảo file txt, file HTML, file đặc tính (.plist), file nhị phân (dùng bộ soạn thảo HEX) và file cơ sở dữ liệu SQLite3.
      • Chỉnh sửa thuộc tính file (tên, owner, group, permission, ngày chỉnh sửa)
      • Chuyển file tới các app khác từ AppStore
      • Mở nhiều loại file từ các app khác với trong iFile
    • Hỗ trợ tối đa 8 tab duyệt file.
    • Hỗ trợ hoàn toàn Dropbox/Box/FTP/WebDAV client: tải lên, xuống, sao chép, di chuyển, đổi tên, hiện hình thu nhỏ vv.
    • Hỗ trợ Toshiba FlashAir® để tải file về máy.
    • Web Server tích hợp với khả năng download và upload vào thư mục hiện tại của Trình duyệt file
    • Hỗ trợ WebDAV để Web Server dùng iFile như ổ đĩa mạng và truy cập được với WebDAV client của iFile.
    • Truyền file qua Bluetooth tới iFile trên máy khác.
    • Truyền file qua Celeste và AirBlue Sharing tới thiết bị khác
    • Nhập nhạc, video và sách vào thư viện iPod qua thư viện Gremlin (yêu cầu cần mua Celeste hoặc YourTube 2/3/HD)
    • Cài đặt các gói Debian (deb)
    • Trình xem file hỗ trợ các loại file sau:
      • File âm thanh: aac, aiff, aif, aifc, amr, alac, caf, m4a, m4r, mp2, mp3, mpga, pcm, snd, wav, wma
      • File văn bản: c, cfg, cnf, conf, cpp, css, h, j, java, js, list, log, m, mm, nib, php, plist, script, sh, strings, txt, xib
      • File Video: mov, mp4, mpv, m4v, 3gp, 3g2, 3gpp2
      • File hình ảnh: gif, jpeg, jpg, png, tiff, tif, bmp, crw, cr2, erf, raf, 3fr, kdc, mef, mrw, nrw, nef, orf, rw2, pef, arw, sr2, svg
      • File tài liệu: rtf, html, htm, pdf, webarchive
      • File của Microsoft: doc, docx, xls, xlsx, ppt, pps, pptx
      • File của Apple iWork: pages, numbers, key
      • Loại file cơ sở dữ liệu SQLite3: sqlite, sqlite3, sqlitedb, db.
      • Các loại file khác: zip, 7z, deb, rar, tar, tar.gz, tar.bz2, tar.Z, tgz, tbz2, gz, bz2, Z, xml

  • Các tính năng hạn chế

    Một số tính năng của iFile chỉ dành cho người dùng đã đăng ký:

    • Tìm kiếm file.
    • Phát danh sách trên Trình phát Audio
    • Hiển thị tên bài hát thay vì tên file cho các file audio.
    • Hiển thị thumbnails thay vì icon hình ảnh mặc định.
    • Chức năng tìm kiếm trong chỉnh sửa văn bản.
    • Hỗ trợ WebDAV của iFile Web Server.
    • Gửi file qua Bluetooth tới iFile trên máy khác.
    • Dùng nhiều tab.
    • Hỗ trợ FTP, box, Dropbox, WebDAV client.
    • Soạn thảo file cơ sở dữ liệu SQLite3.
    • Lưu file nhị phân đã sửa bằng Bộ soạn thảo Hex.
