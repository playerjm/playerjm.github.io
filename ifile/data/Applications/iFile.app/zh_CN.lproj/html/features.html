• iFile 的主要功能

    • 集成的文件浏览器，支持：
      • 同时创建、删除、复制、剪切（移动）、粘贴或压缩多个文件或目录；
      • 重命名文件或目录；
      • 使用书签标记文件或文件夹；
      • 跨层级搜索文件；
      • 使用 Mail 通过电子邮件发送一个或多个文件；
      • 编辑文本、HTML、属性表、二进制（使用十六进制编辑器）以及 SQLite3 数据库等类型的文件；
      • 编辑文件属性（名称、所有者、组、权限、修改日期）。
      • 传递文件给 App Store 应用程序使用；
      • 打开其它应用程序传递给 iFile 的多种类型的文件。
    • 最多可使用 8 个浏览标签；
    • 完整的 Dropbox/Box/FTP/WebDAV 客户端支持：上传、下载、复制、移动、重命名、预览图等等；
    • 支持从东芝 FlashAir® 下载文件到本机；
    • 集成的网页服务器，提供对文件浏览器当前目录中文件的上传和下载功能；
    • 网页服务器的 WebDAV 支持以将 iFile 作为网络硬盘使用或被 iFile 的 WebDAV 客户端访问；
    • 经蓝牙传送文件到其它设备的 iFile；
    • 经 Celeste 和 AirBlue Sharing 传输文件到其它设备；
    • 使用 Gremlin 导入音乐、视频及电子书到 iPod 资源库；
    • 安装 Debian 软件包（deb）；
    • 文件查看器，支持下列类型：
      • 音频文件：aac、aiff、aif、aifc、amr、alac、caf、m4a、m4r、mp2、mp3、mpga、pcm、snd、wav、wma；
      • 文本文件：c、cfg、cnf、conf、cpp、css、h、j、java、js、list、log、m、mm、nib、php、plist、script、sh、strings、txt、xib；
      • 视频文件：mov、mp4、mpv、m4v、3gp、3g2、3gpp2；
      • 图像文件：gif、jpeg、jpg、png、tiff、tif、bmp、crw、cr2、erf、raf、3fr、kdc、mef、mrw、nrw、nef、orf、rw2、pef、arw、sr2、svg；
      • 文档文件：rtf、html、htm、pdf、webarchive；
      • 微软文件：doc、docx、xls、xlsx、ppt、pps、pptx；
      • 苹果 iWork 文件：pages、numbers、key；
      • SQLite3 数据库文件类型：sqlite、sqlite3、sqlitedb、db；
      • 其它文件：zip、7z、deb、rar、tar、tar.gz、tar.bz2、tar.Z、tgz、tbz2、gz、bz2、Z、xml。

  • 受限功能

    以下功能仅适用于已注册者：

    • 文件名搜索；
    • 播放曲目列表；
    • 音频文件显示曲目标题；
    • 图像缩略图；
    • 文本文件内搜索；
    • iFile 网页服务器的 WebDAV 支持；
    • 使用 iFile 经蓝牙发送文件到其它设备；
    • 文件浏览器多标签页；
    • FTP、box、Dropbox 及 WebDAV 客户端支持；
    • 编辑 SQLite3 数据库文件；
    • 十六进制编辑器保存文件。
