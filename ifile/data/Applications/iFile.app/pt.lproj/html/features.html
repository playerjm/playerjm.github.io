• Características do iFile

    • o iFile é explorador de arquivos integrado, com suporte para:
      • Criar, deletar, copiar, cortar (mover), colar ou comprimir (zip) múltiplos arquivos ou diretórios de uma vez
      • Renomear arquivos e diretórios
      • Adicionar marcadores de arquivos e pastas
      • Busca de arquivos
      • Enviar por email, um ou mais arquivos com Mobile Mail da Apple
      • Edição de arquivos de texto, arquivos HTML, arquivos de lista de propriedades, arquivos binários(com Editor Hex) e arquivos do banco de dados SQLite3
      • Editar os atributos dos arquivos (nome, proprietário, grupo, permissão, data de modificação)
      • Passagem de arquivos para outras apps da AppStore
      • Abertura de arquivos de vários tipos de outras apps no iFile
    • Até 8 abas no navegador de arquivos.
    • Completo Dropbox/Box/FTP/WebDAV suporte de clientes: upload, download, copia, movimentação, renomeação, miniaturas, etc.
    • Toshiba FlashAir® Suporte para fazer download de arquivos no sistema local.
    • Servidor Web integrado com capacidade de descarregar (download) ou carregar (upload) arquivos para o diretório selecionado no explorador de arquivos.
    • WebDAV suporte de Servidor Web para usar iFile como uma unidade de rede para ser acessada com o cliente iFile's WebDAV.
    • Transferência de arquivos via Bluetooth para o iFile em outro aparelho.
    • Transferência de arquivos via Celeste e AirBlue Sharing para outro aparelho
    • Importação de músicas, vídeos e livros na livraria do iPod via livraria Gremlin.
    • Instalação de pacotes Debian (deb)
    • Visualizador de arquivos com suporte para os tipos abaixo
      • Sons: aac, aiff, aif, aifc, amr, alac, caf, m4a, m4r, mp2, mp3, mpga, pcm, snd, wav, wma
      • Textos: c, cfg, cnf, conf, cpp, css, h, j, java, js, list, log, m, nib, php, plist, script, sh, strings, txt, xib
      • Vídeos: mov, mp4, mpv, m4v, 3gp, 3g2, 3gpp2
      • Imagens: gif, jpeg, jpg, png, tiff, tif, bmp, crw, cr2, erf, raf, 3fr, kdc, mef, mrw, nrw, nef, orf, rw2, pef, arw, sr2, svg
      • Documentos: rtf, html, htm, pdf
      • Microsoft: doc, docx, xls, xlsx, ppt, pps, pptx
      • iWork da Apple: pages, numbers, key
      • Tipos de arquivo do banco de dados SQLite3: sqlite, sqlite3, sqlitedb, db.
      • Outros: zip, 7z, deb, rar, tar, tar.gz, tar.bz2, tar.Z, tgz, tbz2, gz, bz2, Z, xml

  • Características restritas

    Algumas funções do iFile, somente estão disponíveis para os usuários registrados:

    • Abrir arquivos de outras aplicações, usando o esquema de URL ifile://.
    • Busca de arquivos.
    • Suporte para visualizadores externos.
    • Reproduzir listas de músicas no novo Reprodutor de Audio.
    • Mostrar o título das músicas, ao invés do nome do arquivo de áudio.
    • Mostrar miniaturas, ao invés do ícone padrão para as imagens.
    • Função de busca no editor de textos.
    • WebDAV suporte ao Servidor Web do iFile .
    • Transferência de arquivos via Bluetooth para o iFile em outro aparelho.
    • Uso de múltiplas abas.
    • FTP, box, Dropbox, WebDAV suporte de clientes.
    • Edição de arquivos do banco de dados SQLite3.
    • Salvamento de arquivos binários modificados pelo Editor Hex .
